Authentication

Authenticates user using SIWE

Verifies that the user has the corresponding private key for the Ethereum wallet address they claim to own. Additionally, it generates a new nonce and associates it with the user address for the next sign-in session. If the user doesn't exist, it creates a new user.

Request Body schema: application/json
required
message
string

The message to be signed.

signature
string

The signature of the signed message

Responses

http://localhost:3000/api/auth/siwe
